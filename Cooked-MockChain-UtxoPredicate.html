Predicates over Utxos

type UtxoPredicate a = Maybe a -> Value -> Bool Source #

A UtxoPredicate is a predicate that is used to filter and select utxos in utxosSuchThat. We declare them through a type synonym to create a convenient shallow DSL of predicates.
